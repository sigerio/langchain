# LangChain / LangGraph Learning Repo

本仓库用于按官方教程学习 `LangChain` 与 `LangGraph`。

当前整理约定如下：

- `tutorials/langchain/`：已归档的 LangChain 示例，按主题拆分为基础链路、RAG、历史消息、代理、服务化。
- `tutorials/langgraph/`：后续 LangGraph 学习目录。
- `notes/langchain-note.md`：LangChain 官方教程学习笔记。
- `notes/langgraph-note.md`：LangGraph 官方教程学习笔记。
- `docs/archive/langchain-index.md`：旧文件路径到新归档位置的映射导航。

如果后续询问概念理解、代码写法或功能含义，回答仍以官方教程为依据，并按主题写入对应笔记。
